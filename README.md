# A Simple Python Wrapper For The StreamSB API
# Requirements

# Installation
```sh
pip3 install streamsb
```
# Usage
```python
from streamsb import StreamSB
key = "abc123" #API_KEY from streamsb.com
ob = StreamSB(key)

#Upload Gdrive link
ob.upload_url("https://drive.google.com/v1/testfile")
```
